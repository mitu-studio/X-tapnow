# Security Policy

## Supported Versions

Security fixes are generally applied to the latest main branch.

## Reporting a Vulnerability

Please report vulnerabilities privately and do not open a public issue.

Include:

- A clear description of the vulnerability
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation (if available)

You can report by contacting the maintainers through your private channel.

## Response Process

- We will acknowledge receipt as soon as possible
- We will investigate and confirm scope
- We will prepare and release a fix
- We will coordinate disclosure timing with the reporter
